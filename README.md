# 摄影器材设备出入库管理程序

一个适合本地化部署的轻量级摄影器材管理系统，支持：

- 设备台账维护
- 设备出入库登记
- 手机浏览器扫码登记
- Excel 批量导入
- 二维码生成与下载
- 库内直接修改设备信息
- 列表展示当前借用人

## 运行方式

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

默认会在当前目录自动创建 `equipment.db` SQLite 数据库文件。

## 手机扫码怎么用

1. 让手机和电脑连接到同一个局域网。
2. 在电脑上运行本系统。
3. 用手机浏览器打开 `http://电脑局域网IP:8501`。
4. 进入 **手机扫码** 标签页，选择出库/入库。
5. 用手机摄像头拍摄设备二维码，系统会自动识别设备编码并直接写入电脑数据库。

> 不需要扫码枪，直接用手机访问网页扫码即可。

## 推荐的 Excel 列名

程序会优先识别以下列名：

- `设备编码` / `资产编码`
- `设备名称` / `名称`
- `分类`
- `品牌`
- `型号`
- `序列号`
- `库位`
- `状态`
- `数量`
- `可用数量`
- `负责人`
- `借用人` / `领用人`
- `备注`

## UI 布局

- **总览看板**：查看库存概览、状态分布、近期操作。
- **设备台账**：新增设备、编辑设备信息、维护借用人。
- **出入库登记**：人工录入借出/归还。
- **手机扫码**：手机浏览器直接拍二维码完成出入库。
- **二维码管理**：按设备编码生成二维码并下载 PNG。
- **Excel 导入**：导入已有设备表。
